# smiles-iupac
Jupyter notebook testing the capability of LLM's to convert SMILES strings to accurate IUPAC nomenclature.

## Prerequisites

- Java Runtime for `py2opsin`

Verify installation by running:

```bash
java --version
```

- Huggingface CLI for `datasets`
- API key for your batched-inference-supporting model of choice for `batchata`

## Usage

Check out the [notebook](smiles-iupac.ipynb)!
